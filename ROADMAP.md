# BallistiCore Roadmap

Planned work, grouped by target release. The current shipped version is **1.6.0**
(see the releases table in the [README](README.md)).

## V2

### Licensing & subscription
- **License key system** — license keys carry an **expiry date tied to subscription
  payment**. On expiry, the app shows a **"Subscription expired"** banner and switches
  to **read-only mode**: no new firearm issues, permits, or returns can be created.
  Access to existing records (current register, history, permits, guards, firearms)
  is **not** blocked — operators can still view and export everything they already have.
